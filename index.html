Sobre Mim

Sou desenvolvedor Fullstack, com experiência prática no desenvolvimento de aplicações web modernas utilizando .NET Core (C#), React.js e Next.js. Tenho domínio na criação de APIs REST, integração com bancos de dados SQL e NoSQL, e aplicação de boas práticas como Clean Architecture, CQRS e princípios SOLID. Gosto de escrever código limpo e organizado, buscando sempre entregar soluções escaláveis e eficientes.
